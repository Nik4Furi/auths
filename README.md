## <h2 id="runLocally" >Run Locally </h2>

Clone the project

```bash
  git clone https://github/Nik4Furi/auths
```

Go to the project directory

```bash
  cd auths
```

Go to frontend and backend

```bash
  cd frontend
```

```bash
  cd backend
```

Install Dependies

```bash

yarn 

```


Start the server

```bash
  yarn dev (Run or restart, whenever you save any file(js))
```
